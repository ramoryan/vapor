# Vapor

Vapor will be a template system written in Go, influenced by [pug](https://github.com/pugjs/pug).
Work in progress, first stable version in ~3-6 months.

Come back later for more details!

## Features
	* indent based

	```jade
	div
		div
			div
	```

	```html
	<div>
		<div>
			<div></div>
		</div>
	</div>
	```
